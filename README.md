# 🎨 Тема TT4 Deboart

Дочерняя тема для сайта [deboart.ru](https://deboart.ru) на базе Twenty Twenty-Four.

---

## 🧩 О теме

Тема разработана для визуализации философии Deboart — исследования творческого потенциала через парадигму **«Форма → Содержание»**.

**Ключевые особенности:**
- Полная интеграция с Pods Framework (типы записей: `work`, `debo_lab_entry`)
- Кастомный граф связей между работами
- Таксономии: Форма (6 типов) + Содержание (6 тем)
- Адаптивный дизайн (мобильный, планшет, десктоп)
- Лабораторный стиль: моноширинный, строгий, с микроточками

---

## 🛠️ Технологии

- WordPress 6.6+
- Дочерняя тема Twenty Twenty-Four
- Pods Framework
- Vanilla JavaScript
- Кастомный CSS + theme.json
